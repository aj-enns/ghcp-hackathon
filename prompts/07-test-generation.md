# Exercise 7: Test Generation

## Instructions
1. Open this file in VS Code
2. Open GitHub Copilot Chat (Ctrl+Shift+I or Cmd+Shift+I)

## Prompt

Generate comprehensive tests for this code including:

1. Unit tests covering all functions/methods
2. Edge cases and boundary conditions
3. Mock objects for external dependencies
4. Test data fixtures with realistic examples
5. Code coverage targets and strategy
